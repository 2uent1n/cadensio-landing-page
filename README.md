# Cadensio Landing Page

Marketing website for Cadensio, a side project exploring SaaS development.

## About

This repository contains the landing page for Cadensio, an all-in-one management platform designed for bakeries, pastry shops, and caf√©s. The goal is to help artisans manage their online presence, production, orders, and daily operations in one place.

**Status:** Early-stage side project built as a learning experience. Currently working towards a functional MVP.

## Tech Stack

- **Framework:** Astro
- **UI Components:** React
- **Styling:** Tailwind CSS

The main application uses **Angular** for the frontends (business management platform + customer-facing app) and **NestJS** for the backend.

## Why This Exists

This project serves as a practical playground to learn full-stack SaaS architecture and build production-grade code. The primary focus is on learning by doing and shipping quality software.
